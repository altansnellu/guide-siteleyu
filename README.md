# guide-siteleyu

## 项目简介

`guide-siteleyu` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为一个存放和展示静态 HTML 内容的目录结构。  
适合用于保存个人笔记、示例页面、参考模板或其他无需依赖服务器环境的网页资源。

## 目录说明

仓库根目录下的 HTML 文件通常为独立页面，可直接通过浏览器打开。  
如有子目录，则用于归类存放相关页面及其依赖资源（如 CSS、JS、图片等）。

```
guide-siteleyu/
├── index.html          # 仓库入口页面（可选）
├── example-page.html   # 示例页面
├── notes/              # 笔记类页面
│   ├── note1.html
│   └── note2.html
└── templates/          # 模板类页面
    └── template.html
```

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面内容可能包含文本、样式、脚本等前端资源，均内嵌或同目录引用。
- 归档遵循“一个页面一个文件”或“一个页面一个文件夹”的原则，便于直接打开和迁移。
- 不保证所有页面在移动设备或老旧浏览器上完全兼容。
